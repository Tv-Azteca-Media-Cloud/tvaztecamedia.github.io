§ 1

Entregables del Release 1

El Release 1 consolidó la habilitación completa de la plataforma, desde la arquitectura base hasta el ciclo de vida de la orden. Todos los siguientes módulos fueron implementados, documentados y validados.

  • ✓
    Media Cloud Configuration habilitado — botón "Create Media Plan" activo en OpportunityS1
  • ✓
    Revenue Cloud completo — 5 pasos en orden estricto: Enable Revenue Cloud, Salesforce Pricing, Instant Pricing, Configure at Runtime, BRES1
  • ✓
    9 Permission Sets TVAM_* — creados y refinados con granularidad por rol funcional y técnicoS1S5
  • ✓
    PCM — Catálogo Digital TV Azteca — 12 productos en 3 capas: PCM Comercial, Ad Specification y TargetingS1S3
  • ✓
    9 Media Channels configurados — TV Azteca Uno, ADN40, a+más, Azteca 7, Azteca Noticias y canales digitalesS2
  • ✓
    Line Item Configuration completa — 3 pestañas: Line Item (precio/fechas), Delivery (frecuencia/días) y Targeting (segmentación)S2
  • ✓
    Audience Targeting operativo — Geografía 32 estados, Canales GAM, Segmentos Demográficos con cascadingS2
  • ✓
    Context Service + Context Definitions — TVAM_MediaPlanTransactionCtx vinculado a Pricing ProcedureS3
  • ✓
    DRO Setup — Dynamic Revenue Orchestrator con flujos de fulfillment y amendmentS3
  • ✓
    GAM Integration end-to-end — primer aprovisionamiento real a Google Ad Manager vía REST API OAuth 2.0S3
  • ✓
    In-Flight Amendments habilitados — modificación de órdenes activas durante el vuelo sin detener la entregaS4
  • ✓
    Nomenclaturas 2026 formalizadas — convenciones para CPM, Patrocinio y Línea de PedidoS4
  • ✓
    Etapas de Cotización — Venta Directa — Borrador → En Autorización → En Presentación → Aceptada / DenegadaS4
  • ✓
    Price Waterfall con BRE activo — descuentos automáticos por Ad Creative Size Type y Ad Space SpecS5
  • ✓
    Etapas de la Order — ciclo de vida completo — 5 estados con rutas incluyendo el AmendmentS5
